# Авто Рынок

Автомобильная доска объявлений для Бишкека и Кыргызстана. Покупка и продажа автомобилей, запчастей, услуг и спецтехники.

## О проекте

**Авто Рынок** — это современная платформа для размещения и поиска автомобильных объявлений. Сайт разработан с упором на удобство использования, быструю навигацию и адаптивный дизайн для всех устройств.

### Ключевые особенности

- **Категории объявлений**: легковые автомобили, мотоциклы, коммерческий транспорт, спецтехника, запчасти, услуги, бизнес, аукционы и номера
- **Адаптивный дизайн**: корректное отображение на десктопах, планшетах и мобильных устройствах
- **Современный UI**: градиенты, карточки, плавные анимации и интуитивная навигация
- **Быстрый поиск**: поиск по сайту с мгновенными результатами
- **Страницы контактов и правил**: прозрачная информация о связи и правилах использования

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — стили, градиенты, адаптивность, анимации
- **JavaScript** — интерактивность (поиск, аккордеоны, кнопка наверх)
- **SVG** — иконки без внешних библиотек

## Как запустить

### Локальный запуск

1. Клонируйте или скачайте проект
2. Откройте любой файл `.html` в браузере (например, `index.html`)
3. Сайт работает без сервера — все файлы статические

### Рекомендуемые браузеры

- Chrome/Edge (последние версии)
- Firefox (последние версии)
- Safari (последние версии)

## Адаптивность

Сайт автоматически адаптируется под ширину экрана:

- **Десктоп (>1100px)**: три колонки (левая реклама, центр, правая реклама)
- **Планшет (900–1100px)**: две колонки
- **Мобильный (<900px)**: одна колонка, скрытие рекламы
- **Мобильный (<600px)**: категории в 2 колонки, оптимизированные отступы
